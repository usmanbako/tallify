Why Tallfind exists

A directory built by people who spent too many afternoons returning pants.

Tallfind is a curated, hand-reviewed directory of clothing brands that actually carry tall sizes — not a "maybe if you're lucky" size run with one or two extended options buried on page six. Every store in the directory has been checked for real, documented tall sizing: minimum inseams, extended top lengths, or dedicated tall product lines.

The site exists because shopping for tall clothes online is miserable. Search engines surface the same handful of big brands, influencer roundups rotate through the same affiliate bait, and most so-called "tall" sections cap out at 34″ inseam or large-only tops. If you're 6'4" and looking for a 38″ inseam jean, the internet doesn't actually help you — it wastes your afternoon.

So we built a directory that does: every store verified, every size range confirmed, updated regularly. If a brand drops tall sizing, it gets removed. If a brand adds tall sizing, it gets added. No SEO spam, no gated signup, no 47 pop-ups.

What we do, in plain English

  • Maintain a list of ~86 (and growing) tall-friendly brands for men and women.
  • Check each store's actual size range — not their marketing copy.
  • Tag stores by inseam, by category (tops / bottoms), by tall-specific vs. mainstream-with-tall-section.
  • Keep the whole thing free to use.

Who runs this

Tallfind is an independent project, not a VC-backed content farm. The dataset started from a widely-shared Reddit thread by u/wildthingking, which we expanded, verified, and structured into a live directory one store at a time.

If you want to contribute — a missing store, incorrect info, or general feedback — the feedback form is the fastest path. For new store submissions, use the Submit a Store form.

How we stay honest

We're building out affiliate relationships with some of the brands listed here so the site can pay for itself, but inclusion is never paid for. Read the full How We Review page for the criteria and the affiliate disclosure for how monetization works.
